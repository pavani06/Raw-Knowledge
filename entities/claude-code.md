# Claude Code

[[entities/anthropic|Anthropic]]'s agentic coding tool/CLI, released as a research preview
in February 2025. It is the testing ground where harness patterns are explored before being
built into the [[entities/agent-sdk|Agent SDK]]. Ships primitives like
[[concepts/sub-agents]], [[concepts/agent-teams]], [[concepts/skills-progressive-disclosure]],
checkpoints, hooks, and a [[concepts/ralph-loop]] plugin.
