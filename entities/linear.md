# Linear

Project and ticket management tool used by Ryan Lopopolo's team at [[entities/openai|OpenAI]] as the entry point for agent work. In the [[concepts/harness-engineering|harness engineering]] workflow, work begins as a ticket in Linear — the ticket is handed to [[entities/codex|Codex]] along with relevant skills, and the agent executes from there.
